# Задача: подробный рецепт

Тебе передадут блюдо из меню (название, гарниры). Напиши подробный пошаговый
рецепт на 2 порции, используя продукты и технику из контекста семьи.

## Формат ответа

Верни СТРОГО валидный JSON, БЕЗ markdown-фенсов, БЕЗ пояснений:

```
{
  "content_md": "Текст рецепта в формате Telegram HTML (см. ниже)",
  "ingredients": [
    {"name": "Куриные бёдра", "quantity": "500", "unit": "г", "store": "Makro"}
  ],
  "prep_minutes": 30
}
```

## Формат content_md (Telegram HTML)

Сообщение пойдёт в Telegram с parse_mode=HTML. Используй ТОЛЬКО эти теги:
`<b>`, `<i>`. Никаких `<h1>/<h2>/<p>/<br>/<ul>/<li>`, никаких markdown-заголовков
(`#`, `##`), таблиц (`|`), разделителей (`---`), цитат (`>`), КБЖУ-блоков.

Структурируй так (пример — следуй точно этому шаблону):

```
<b>🍳 Свиная вырезка в airfryer + гречка + салат</b>
<i>2 порции · активная готовка ~35 мин</i>

<b>Ингредиенты</b>

<b>Свиная вырезка</b>
• Вырезка — 450–500 г
• Соевый соус — 2 ст.л.
• Паприка — 1 ч.л.

<b>Гречка</b>
• Гречка — 160 г
• Вода — 320 мл

<b>Салат</b>
• Огурцы — 2 шт.
• Помидоры — 2 шт.

<b>Шаги</b>

<b>1. Маринуем вырезку</b> <i>(5 мин)</i>
Обсушить вырезку. Смешать маринад: соевый соус, паприка, оливковое масло, перец. Натереть со всех сторон, оставить мариноваться пока греется airfryer.

<b>2. Варим гречку</b> <i>(параллельно)</i>
Промыть, засыпать в Instant Pot 1:2 с водой, щепотка соли. Pressure Cook 6 мин + естественный сброс 5 мин. Добавить масло, перемешать.

<b>3. Готовим вырезку</b> <i>(20 мин)</i>
Airfryer 200°C, 18–20 мин с переворотом на середине. Внутренняя температура 65–68°C. Дать отдохнуть 5 минут перед нарезкой.

<b>4. Салат</b> <i>(5 мин)</i>
Огурцы и помидоры нарезать, заправить оливковым маслом, посолить, поперчить.

<b>💡 Совет</b>
Вырезка — нежирный отруб, не передерживай. 65–68°C внутри = сочное мясо.
```

## Требования

- content_md: полностью на русском.
- Каждая секция отделена пустой строкой. Шаги — одним абзацем, без вложенных списков.
- Не пиши КБЖУ — не нужно.
- Длина — компактно, но содержательно (ориентир 1500–2500 символов).
- ingredients: каждый ингредиент с количеством на 2 порции. Поле "store" одно из:
  "Makro", "Villa Market", "Lotus's", "7-Eleven", "Кладовка" (последнее — для
  базовой кладовки, не нужно покупать).
- prep_minutes — реалистичная оценка АКТИВНОЙ готовки (не больше 40 для основного блюда).
- НЕ используй лук в салатах. НЕ используй индейку. НЕ используй чеснок в блюдах.
